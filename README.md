# Attendance

## Usage 
 
To take the face samples
```bash
python3 samplefecth.py
```

To train the samples 
```bash
python3 FaceTraining.py
```

To open up Face Recognition to record the attendence
```bash
python3 Face_Sets.py
```

To open up the website
```bash
python3 run.py
```
